# Saudi Interview Coach

Next.js scaffold for the Saudi Interview Coach MVP described in `Saudi_Interview_Coach_PRD.md`.

## Scripts

```bash
npm run dev
npm run build
```

## Environment

Copy `.env.example` to `.env.local`, then add your OpenRouter API key:

```bash
OPENROUTER_API_KEY=your_openrouter_api_key_here
OPENROUTER_MODEL=openai/gpt-4o-mini
```

`OPENROUTER_MODEL` is optional. Change it to any OpenRouter chat model you want to use.
